## VS Code

Open `Preferences: Open Keyboard Shortcuts(JSON)`, binding key to run test task

```json
{
  "key": "f6",
  "command": "workbench.action.tasks.test"
}
```

Add this config to `settings.json`

```json
{
  "C_Cpp.default.includePath": ["headers"]
}
```
